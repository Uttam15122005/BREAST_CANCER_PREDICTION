## 📌 Project Overview

This project is a **Machine Learning-based web application** that predicts whether a tumor is **benign or malignant** using medical data.

It helps in **early detection of breast cancer**, improving decision-making in healthcare systems.

---

## 🎯 Features

- 🔍 Predict cancer in real-time
- 📊 Uses 30 tumor features
- 🧠 ML model with preprocessing (StandardScaler)
- 🌐 Flask-based web app
- 🎨 Attractive UI design

---

## 🧪 Machine Learning Model

- Algorithm: Logistic Regression  
- Dataset: Breast Cancer Wisconsin Dataset  
- Preprocessing: StandardScaler  
- Accuracy: ~95%
